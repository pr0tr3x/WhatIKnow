Iptables

Execute command without internet access

Create new group

sudo addgroup drop-connect
Create new group 
user@user:~$ sudo addgroup drop-connect
Adding group `drop-connect' (GID 1001) ...
Done.

Add iptables rule

For ipv4 

sudo iptables -A OUTPUT -m owner --gid-owner drop-connect -j DROP

For ipv6 

sudo ip6tables -A OUTPUT -m owner --gid-owner drop-connect -j DROP

Add rules 
user@user:~$ sudo iptables -A OUTPUT -m owner --gid-owner drop-connect -j DROP
user@user:~$ sudo ip6tables -A OUTPUT -m owner --gid-owner drop-connect -j DROP
user@user:~$ sudo iptables-save

Restoring iptables rules after reboot

Install iptables-persist package

sudo apt install iptables-persistent

Any currently erected iptables rules will be saved to the corresponding /etc/iptables/rules.v4 and /etc/iptables/rules.v6.

Updating files with rules

sudo iptables-save > /etc/iptables/rules.v4
sudo ip6tables-save > /etc/iptables/rules.v6
